Classic Literature Data

The data on this webpage is provided by the Classics JSON Library of the CORGIS Dataset Project. It contains information about the 1000 most popular books by download on Project Gutenburg. Project Gutenburg is a digital library of over 56,000 ebooks that have fallen into the public domain. Created by Michael S. Hart in 1971, it is the oldest digital library.


Data about these novels is available by title, genre, and reading level. Genre is dislayed using the Library of Congress Classification System. Reading level is sorted by grade, and calculated using the Coleman Liau Index, Gunning Fog System, Automated Readability Index, and Flesch-Kincaid Grade Level, all of which measure years of formal education needed to read and understand a given work.
